!2) Post запрос
!3) тело запроса: 
{
    "expression": "выражение, которое ввёл пользователь"
}
!4) тело ответа:
{
    "result": "результат выражения"
}
Код 200, если выражение вычислено успешно
!5) тело ответа:
{
    "error": "Expression is not valid"
}
Код 422, если входные данные не соответствуют требованиям приложения — например, кроме цифр и разрешённых операций пользователь ввёл символ английского алфавита
6) тело ответа:
{
    "error": "Internal server error"
}
Код 500 в случае какой-либо иной ошибки («Что-то пошло не так»)
